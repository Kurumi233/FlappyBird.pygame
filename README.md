# Flappy Bird python version

### Requirements

- pygame

```bash
pip install pygame
```

### Usage

- 点击鼠标 / 键盘方向键 控制起飞
- 点击右上角PAUSE / 键盘空格键 控制暂停/继续
- 结束后点击重新开始
